# Restrict Admin Access Plugin

## Description
Restrict Admin Access is a WordPress plugin that restricts access to the admin panel for selected user roles and hides the admin bar.

## Installation
1. Download the plugin zip file from the [Restrict Admin Access Plugin](https://martinenrique.com/restrict-admin-access/) page.
2. Log in to your WordPress admin panel.
3. Go to **Plugins** > **Add New**.
4. Click on the **Upload Plugin** button and select the zip file you downloaded.
5. Click **Install Now** and then **Activate Plugin**.

## Configuration
1. After activating the plugin, a new field will appear under **Settings** > **General**.
2. Scroll down to the **Restrict Admin Access** section.
3. Select the user roles you want to restrict from accessing the admin panel and hide the admin bar.
4. Click **Save Changes**.

## Functionality
Once the plugin is configured, users with the selected roles will be restricted from accessing the admin panel. They will be redirected to the homepage if they try to access it. The admin bar will also be hidden for these restricted users.
